## Usage

- 用法： `./test.py` 
    - 选择下载项目时可以输入多个空格分隔的数字，非法输入将被丢弃
    - 命令行参数：
		- `-d` 删除文件夹
		- `-n` 以新的文件名保存
		- `-s` 跳过 （默认）
		- `-o` 覆盖文件名重复的文件
-  `*.m3u8` ：下载 [`vlc player`](https://www.videolan.org/vlc/) 本地播放，或安装 `chrome` 插件 [`Native HLS Playback`](https://chrome.google.com/webstore/detail/emnphkkblegpebimobpbekeedfgemhof) 在浏览器内播放
